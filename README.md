## 🔧 Prerequisites

- Node.js 18+ (React 18.3.1)
- npm or yarn
- Git

## 📥 Installation

```bash
# Clone the repository
git clone https://github.com/Dan-Duran/learn-bash.git

# Navigate to the directory
cd learn-bash

# Install dependencies
npm install 
# or
yarn install

# Run the development server
npm run dev
# or
yarn dev
```
